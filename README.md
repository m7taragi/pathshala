# Pathshala

**Pathshala** is a central portfolio hub documenting my web development journey. It serves as a unified platform to showcase various projects, each exploring unique aspects of modern front-end design, layout strategies, interactivity, and utility.

## 🚀 Overview

The frontend acts as a portfolio gallery showcasing the following projects:
- **Project 1: Glass UI** - An exploration of transparency and blur effects in modern dashboards using HTML/CSS.
- **Project 2: Flex-Grid** - A complex responsive landing page built purely with CSS Grid systems.
- **Project 3: Mini Games** - Small logic-based interactive games developed using basic JavaScript and DOM events.
- **Project 4: Task Tracker** - A lightweight productivity tool for managing daily coding habits.
- **Project 5: Type-Study** - A focused look at typography, hierarchy, and reading flow on the web.

Additionally, the project structure incorporates a Node.js backend environment with core dependencies (Express, Mongoose, JWT, bcryptjs) to support full-stack application development and authentication workflows.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend Environment:** Node.js, Express.js
- **Database (via dependencies):** MongoDB (Mongoose)
- **Security & Auth:** bcryptjs, jsonwebtoken, cors, dotenv
- **Design:** Custom Dark/Light mode theme, responsive grid layouts

## 📁 Project Structure

- `index.html`: The main entry point and portfolio dashboard.
- `style.css`: Core styling including the dark/light theme toggle and responsive grid layouts.
- `script.js`: Frontend logic for interactions and theme toggling.
- `HTML/`, `Project2/` to `Project5/`: Individual project directories showcasing specific skills.
- `app/`, `cave-core/`, `webforms/`: Backend and full-stack integration directories.
- `package.json`: Node dependencies and project configuration.

## ⚙️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/m7taragi/pathshala.git
   cd pathshala
   ```

2. **Install Dependencies:**
   Ensure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **View the Portfolio:**
   Simply open `index.html` in your web browser, or use an extension like VS Code Live Server to view the frontend interface.
